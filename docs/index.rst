PyKaldi
-------

Welcome to `PyKaldi <https://github.com/pykaldi/pykaldi>`_ documentation.
PyKaldi is a Python wrapper for `Kaldi <http://kaldi-asr.org>`_. It aims to
bridge the gap between Kaldi and all the nice things Python has to offer.


Features
--------

* Near-complete coverage of Kaldi C++ API

* First class support for Kaldi and OpenFst types in Python

* Extensible design

* Open license

* Extensive documentation

* Thorough testing

* Example scripts

* Support for both Python 2.7 and 3.5+
